# Line Follower Robot 🤖

## 📌 Description
This project is a Line Follower Robot built using ESP32, IR sensors, and an ultrasonic sensor for obstacle avoidance.

## 🛠 Components Used
- ESP32
- L298N Motor Driver
- IR Sensors (3)
- Ultrasonic Sensor (HC-SR04)
- BO Motors
- Wheels + Chassis

## ⚙️ Features
- Line Following using IR Sensors
- Obstacle Detection using Ultrasonic Sensor
- Smooth Movement using PID Control
